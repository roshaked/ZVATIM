# תרגול ניהול ופיתוח צוותים - מבוסס חומר מקור

אתר סטטי בעברית לתרגול קורס ניהול ופיתוח צוותים. התוכן מבוסס רק על קבצי הקורס שסופקו: מצגות, דף לסטודנט ומפת סוגי צוותים.

## מה יש באתר

- שאלות אמריקאיות עם תשובות מעורבבות ומשוב מיידי.
- מילון מושגים שבו כל מושג באנגלית מופיע עם פירוש בעברית.
- קישורים מכל מושג אל עמוד חומר המקור, כולל ציון מצגת/שקופית.
- שאלות תרחיש עם דרך פעולה מומלצת ונימוק.
- עיצוב RTL מותאם לשיתוף דרך GitHub Pages.
- קבצי המצגות עצמם אינם מפורסמים בריפו הציבורי.
