# תרגול ניהול ופיתוח צוותים

אתר סטטי בעברית לתרגול קורס ניהול ופיתוח צוותים.

## מה יש באתר

- שאלות אמריקאיות עם תשובות מעורבבות ומשוב מיידי.
- מילון מושגים שבו כל מושג באנגלית מופיע עם פירוש בעברית.
- קישורים מכל מושג אל חומר המצגת הרלוונטי.
- שאלות תרחיש עם דרך פעולה מומלצת ונימוק.
- עיצוב RTL מותאם לשיתוף דרך GitHub Pages.

## העלאה ל-GitHub Pages

האתר בנוי כקבצים סטטיים בלבד:

- `index.html`
- `materials.html`
- `styles.css`
- `app.js`

ב-GitHub Pages יש לבחור פרסום מהענף הראשי ומתיקיית השורש.
